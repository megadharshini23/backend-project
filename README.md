# 🔢 Counter App with Dark Mode Toggle

A clean and interactive beginner-friendly web application built using **HTML, CSS, and Vanilla JavaScript**.  
This project demonstrates DOM manipulation, event handling, and theme switching in a real-world mini UI.

---

## 📌 Live Demo
👉 https://github.com/megadharshini23/github.io/backend-project

---

## 🚀 Features

- ➕ Increase counter value dynamically  
- ➖ Decrease counter value dynamically  
- 🔁 Reset counter to zero  
- 🌙 Toggle Dark / Light mode  
- 🎨 Dynamic text color based on counter value:
  - 🟢 Positive → Green  
  - 🔴 Negative → Red  
  - ⚫ Zero → Black  
- ⚡ Smooth and responsive UI interactions

## ⚙️ How It Works

- A variable `count` stores the current state of the counter  
- Button clicks update the value dynamically  
- UI is updated using JavaScript DOM manipulation  
- `updateColor()` function changes text color based on value  
- Dark mode is handled using `body.classList.toggle("dark")`
